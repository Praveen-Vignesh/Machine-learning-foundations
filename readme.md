# Machine Learning Foundations

This repository serves as a structured documentation of my machine learning studies. It contains mathematical notes, algorithm implementations in Python, and structured coursework.

## Overview

The repository is organized into specific domains to separate theory from implementation:

*   **Math Foundations:** Notes, proofs, and exercises focusing on calculus and linear algebra required for algorithm     comprehension.
*   **Source Code:** Reusable Python modules, data loaders, and evaluation metrics.
*   **Notebooks:** Exploratory data analysis and model testing environments.
